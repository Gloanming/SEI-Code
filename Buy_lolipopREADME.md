## 买棒棒糖

已知在思远哥哥的糖果商店，一根棒棒糖的价格是 6 元 7 角，而你有 a 元 b 角**硬币**，你最多能买多少根棒棒糖？
## 输入格式
输入只有一行两个**非负**整数，以空格分隔，分别表示 a 和 b 。
## 输出格式
输出一行一个整数，表示最多能买多少根棒棒糖。
## 输入输出样例
输入1
```
20 3
```
输出1
```
3
```
输入2
```
19 15
```
输出2
```
3
```
