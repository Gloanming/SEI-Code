# 增量元素之间的最大差值



在一行中输入若干个整数，视为数组nums（长度为n），
请你计算 nums[j] - nums[i] 能求得的 **最大差值** ，其中 0 <= i < j < n 且 nums[i] < nums[j] 。

返回 **最大差值** 。如果不存在满足要求的 i 和 j ，返回 -1 。



**小提示**：想过这道题的test，你可能需要知道print函数更多的一些细节哦ʕ •ᴥ•ʔ



#### 样例1

```
input: 7 1 5 4
output: 4
```



#### 样例2

``` 
input: 9 4 3 2
output: -1
```
